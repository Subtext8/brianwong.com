# Claude Code

Claude's coding interface/tool for vibe coding.

## From [[Wiki/Sources/Ways-to-Use-AI-in-Daily-Life|Ways-to-Use-AI-in-Daily-Life]]
- Vibe code (advanced).
- Presentations, dashboards, interactive.

## From [[Wiki/Sources/Using-Claude-Code-The-Unreasonable-Effectiveness-of-HTML|Using Claude Code The Unreasonable Effectiveness of HTML]]
- Claude Code can ingest broad context (codebase, MCPs, git history) to generate rich HTML artifacts.
- It is used to create specs, code reviews, and custom editing interfaces.
- It allows for two-way interaction, enabling users to adjust designs or parameters and export the results.
