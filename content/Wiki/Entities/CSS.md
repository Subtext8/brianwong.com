# CSS

## Description
Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language like HTML.

## From [[Wiki/Sources/Using-Claude-Code-The-Unreasonable-Effectiveness-of-HTML]]
- CSS is used to style HTML artifacts generated by AI agents, allowing for better visual clarity and design.
- It enables the creation of responsive layouts and custom UI components.
