# Perplexity

AI search engine specialized for research: ChatGPT + search with citations. Deep research mode, curates news from multiple sources (incl. social). API for bulk workflows (e.g., research tools from Google Sheets).

## From [[Sources/Ways-to-Use-AI-in-Daily-Life|Ways-to-Use-AI-in-Daily-Life]]
- Targeted searches better than Google (e.g., investments for Canadian in Hong Kong earning USD).
- Deep research for in-demand careers, compare medicine models.
- Curate news.
- API workflows: upload Sheets, summarize tools in bulk.
