# Claude

Anthropic's LLM used in AI products.

## From [[Wiki/Sources/10-AI-Concepts|10-AI-Concepts]]
- Example LLM (ChatGPT, Gemini).

## From [[Wiki/Sources/Using-Claude-Code-The-Unreasonable-Effectiveness-of-HTML|Using Claude Code The Unreasonable Effectiveness of HTML]]
- Claude is increasingly being used to generate HTML artifacts instead of Markdown for better information density and interactivity.
- It can organize structure visually with tabs, illustrations, and links.
- Claude can sketch out designs in HTML and then write them in other languages like React or Swift.

## From [[Wiki/Sources/Post-by-karpathy-on-X|Post by @karpathy on X]]
- Andrej Karpathy highlights Claude's ability to generate HTML as a key step in the evolution of AI output formats.
