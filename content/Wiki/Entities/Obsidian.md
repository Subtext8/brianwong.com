# Obsidian

Obsidian: Local-first knowledge base tool with wikilinks, graph view, plugins.[[Wiki/Sources/The Kaparthy LLM Wiki|The Kaparthy LLM Wiki]]

## Role in LLM Wiki
- **IDE**: LLM edits markdown files; user browses live.
- **Graph view**: Visualize page connections, detect orphans/hubs.
- **Ingestion**: Web Clipper → raw sources; hotkey download images.
- **Plugins**:
  - Marp: Markdown slides from wiki.
  - Dataview: Query frontmatter.
- **Tips**: Fixed attachment folder (e.g. raw/assets/).
