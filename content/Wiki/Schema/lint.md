# Lint the Wiki

**Copy to Copilot:**

Lint the wiki:

1. Read [[TOC]], [[log]]

2. Orphans: obsidianLinks orphans path:"Wiki" verbose:true

3. Recent activity: tail log

4. Search contradictions: localSearch queries like "claim1 NOT claim2"

5. Stale pages: compare date vs recent ingests

6. Suggestions:
   - Merge similar pages
   - New pages for missing concepts
   - New sources to resolve gaps
   - Edits for consistency

Propose actions: e.g. "Delete orphan [[page]]?" or editFile.
