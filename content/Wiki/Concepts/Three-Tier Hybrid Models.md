# Three-Tier Hybrid Models

Strategic hybrid AI infra: cloud (elasticity), on-prem (consistency/security), edge (latency/privacy).

## Overview
Response to cloud-only costs; enables inference economy scaling.

## Details/Key Facts
| Tier | Role |
|------|------|
| Cloud | Peak loads/training |
| On-Prem | High-volume TCO |
| Edge | Real-time (drones/devices) |
