# Persistent Wiki

**Core idea**: A "persistent, compounding artifact" where LLM integrates new sources into existing structure, keeping knowledge current.[[Wiki/Sources/The Kaparthy LLM Wiki|The Kaparthy LLM Wiki]]

## Features
- Pre-built cross-references and synthesis.
- Flags contradictions, updates summaries.
- Evolves with ingests and queries.
- LLM owns all writes; human reads/steers.

## Why Superior to RAG
- Avoids repeated synthesis.
- Maintenance scales with LLM (touches multiple pages per ingest).
