# Stanford-STORM

## Summary
[[Wiki/Entities/Stanford-STORM|Stanford-STORM]] is an LLM-based system designed to automate the creation of Wikipedia-like articles from scratch using Internet search. It breaks the process into a pre-writing stage (research and outline generation) and a writing stage (full-length article generation with citations). [[Wiki/Entities/Co-STORM|Co-STORM]] extends this by enabling human-AI collaboration, featuring a collaborative discourse protocol, turn management, and a dynamic mind map to build a shared conceptual space.
